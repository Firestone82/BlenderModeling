# BlenderModeling

> **VŠB-TUO** — School project · 3D Modeling (MGA)

![Blender](https://img.shields.io/badge/Tool-Blender-orange)

## About

A collection of 3D modeling and rendering assignments completed for the MGA (Modeling and Graphics Applications) course at VŠB-TUO. Projects range from basic object modeling (furniture, everyday items) to more advanced work including terrain generation, architectural models, and baked lighting renders.

## Contents

| Assignment | Description |
|------------|-------------|
| Chair, Table, Cup, Book | Basic object modeling |
| House | Architectural modeling |
| Terrain | Procedural terrain generation |
| Maze structure | Complex geometry |
| Gears | Mechanical modeling |
| Baking tests | Texture baking and light rendering |

Each folder contains the `.blend` source file and exported PNG renders.

## Getting started

1. Install [Blender 3.x or later](https://www.blender.org/download/).
2. Clone the repository:
   ```bash
   git clone https://github.com/Firestone82/BlenderModeling.git
   ```
3. Open any `.blend` file directly in Blender.
